[[Internal-API]]
== 9. 내부 연동 (Internal) API
이 API는 RabbitMQ(또는 기타 메시지 브로커)의 Webhook 기반 인증/인가를 위해 사용됩니다.
프론트엔드 클라이언트가 아닌, 시스템 내부에서만 호출되는 `x-www-form-urlencoded` 형식의 API입니다.
반환값은 순수 문자열인 `allow` 또는 `deny` 입니다.

=== 9.1. 브로커 연결 인증 (Authentication)
유저의 JWT 토큰 또는 기기의 Secret을 확인하여 브로커 연결을 허용하거나 거부합니다.
